== Description ==

This plugin will help you to have ajaxified comments on your site. The comment submission and pagination, also inline reply will be in ajax once you enable this theme.

A few notes about the sections above:
* This plugin is tested with the default theme twentyeleven. So, you might have to change some of the ID's  or Classes in javascript file for this plugin to work.

== Installation ==

Upload the ajax-comment plugin to your blog, Activate it
